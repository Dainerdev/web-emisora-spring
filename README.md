# Aplicación Web - Gestor de Emisoras

Aplicación web como proyecto académico, orientada a la gestión de emisoras y la administración de la información asociada. El proyecto fue diseñado para fortalecer habilidades en desarrollo backend, lógica de negocio y estructuración de aplicaciones web.

## Tecnologías
- Java
- Spring
- HTML, CSS, JavaScript
- MySQL

## Funcionalidades
- Registro y gestión de emisoras.
- Administración de personas asociadas a una emisora.
- Relación y control de información entre emisoras y productoras.
- Operaciones básicas de gestión de datos (crear, consultar, actualizar).
